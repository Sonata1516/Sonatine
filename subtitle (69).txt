В предыдущем видео вы узнали о Git и GitHub. Прежде чем продолжить работу с этим видео, зарегистрируйтесь для учетной записи GitHub и войдите в систему. Начнем с создания нового репозитория. Нажмите +, затем выберите Создать репозиторий. Чтобы создать новый репозиторий, вам необходимо указать следующие сведения: дать новому репозиторию имя; дополнительно добавить описание репозитория; выбрать видимость репозитория - будь он публичным или частным; и выбрать опцию Инициализировать этот репозиторий с помощью файла readme. Затем нажмите кнопку Создать репозиторий. Теперь вы будете перенаправлены в созданный вами репозиторий. Корневая папка вашего репозитория указана по умолчанию и имеет только один файл Readme.md. Теперь пришло время отредактировать файл readme. Вы можете сделать это в своем браузере. Просто нажмите на карандаш, чтобы открыть онлайн-редактор, и вы можете изменить текст readme. Чтобы сохранить изменения в репозитории, их необходимо зафиксировать. После внесения изменений прокрутите вниз до раздела «Зафиксировать изменения». Добавьте сообщение фиксации и, при необходимости, добавьте описание, а затем нажмите кнопку Зафиксировать изменения. « commit changes» используется для сохранения изменений в репозитории. Вернитесь на главный экран, щелкнув ссылку на имя репозитория. Обратите внимание, что файл readme обновляется и проверьте изменения. Давайте узнаем, как создать новый файл с помощью встроенного веб-редактора, предоставленного GitHub, который работает в браузере. Нажмите «Добавить файл», а затем «Создать новый файл», чтобы создать новый файл. Чтобы создать файл python с именем firstpython.py. Сначала укажите имя файла. Затем добавьте комментарий, описывающий ваш код, а затем добавьте код. После завершения, зафиксируйте изменения в репозитории. Вы можете видеть, что ваш файл теперь добавлен в репозиторий, и список репозитория показывает, когда файл был добавлен или изменен. Когда вам нужно изменить файл, вы можете изменить его снова. Щелкните имя файла, затем щелкните значок карандаша, внесите изменения и внесите изменения. Вы также можете загрузить файл из вашей локальной системы в репозиторий. На главном экране репозитория нажмите Добавить файл и выберите параметр Загрузить файлы. Нажмите «Выбрать файлы» и выберите файлы, которые вы хотите загрузить из вашей локальной системы. Процесс загрузки файлов может занять короткое время, в зависимости от того, что вы загружаете. После завершения загрузки файлов нажмите кнопку «Commit Changes». Теперь в репозитории отображаются загруженные файлы. В этом видео вы научились создавать репозиторий, редактировать файлы и фиксировать изменения с помощью веб-интерфейса.
